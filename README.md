## 🌸 プロジェクト概要

### 商品情報
- **商品名**: 桜の泉NMN
- **製造**: 占部大観堂製薬株式会社
- **カテゴリ**: 健康食品・サプリメント（若返り・エイジングケア）

### 主要成分
1. **NMN（ニコチンアミド・モノヌクレオチド）** - 20,000mg/瓶
   - サーチュイン遺伝子活性化
   - 細胞レベルからの若返りサポート
   - 体内修復機能の向上

2. **焼成サンゴCa水素パウダー** - 沖縄県産100%
   - 36時間持続する水素効果
   - 強力な抗酸化作用
   - 健康効果と若々しさの維持

3. **PQQ（ピロロキノリンキノン）** - 600mg/瓶
   - エネルギービタミン
   - ミトコンドリア新生促進
   - ビタミンC・Eより強力な抗酸化力

### サポート成分
- レスベラトロール含有赤ワインエキス（フランス・ボルドー産）
- ビタミンE
- αリポ酸
- コエンザイムQ10

## 🔧 カスタマイズ方法

### 色の変更
`css/style.css`のCSS変数を編集：
```css
:root {
    --primary-pink: #FFB7C5;        /* メインのピンク色 */
    --primary-gold: #D4AF37;        /* アクセントのゴールド */
    --secondary-cream: #FFF8F0;     /* 背景のクリーム色 */
}
```

### テキストの変更
`index.html`の各セクション内のテキストを直接編集してください。

### アニメーション速度の調整
`js/main.js`のAOS設定を編集：
```javascript
AOS.init({
    duration: 800,  // アニメーション時間（ミリ秒）
    once: true,     // 一度だけアニメーション
    offset: 100,    // トリガーのオフセット
});
```
